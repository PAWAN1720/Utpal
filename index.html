GJV Translator

What's new in this translator?

GJV Translator is an Assamese translation app designed especially for Assamese people (for now), with more features planned in the future.

What’s new in this version is that it now includes Assamese grammar support, grammar learning tools, and Assamese grammar games to make learning more fun and interactive. It also offers parsing features and a collection of important and uncommon words to help users understand the language more deeply.

Along with translation, the app also has a built-in music system, so you can enjoy listening while using it. There’s also a simple in-app guide to help you get started easily.

No login is required, and the app is lightweight and easy to download.

Published by Gyaanpeeth Jatiya Vidyalaya
